# âœï¸ Text

The format of texts

The plugin employs the `MiniMessage` ( [https://docs.advntr.dev/minimessage/format.htmlarrow-up-right](https://docs.advntr.dev/minimessage/format.html)) format for text representation. This system provides a rich and flexible way to format text with colors, styles, and other features. However, if you're more comfortable with or require the old color format (using the '&' symbol), it's still supported.

#### [hashtag](\#legacy-color-code-support)    Legacy Color Code Support:

Here's how it should look in the `config.yml`:
